# Jetpack Compose Todo App

Simple Todo list Android app built to practice **Jetpack Compose** fundamentals and Compose state management.

## Features

- Add, complete, and delete todo items
- Reactive UI driven by `remember` and `mutableStateOf`
- Material 3 theming

## Tech Stack

- **Kotlin** + **Jetpack Compose**
- **Material 3**
- **ViewModel**

## Getting Started

1. Open in Android Studio
2. Run on an emulator or device (Android 5.0+)
